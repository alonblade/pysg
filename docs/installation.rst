.. _installation:

============
Installation
============

Install the latest stable version of pysg with :command:`pip`:

.. code-block:: sh

    $ pip install pysg --extra-index-url https://api.packagr.app/JpLhv7UqD/ --disable-pip-version-check

Check version:

.. code-block:: py

    import pysg
    pysg.__version__
