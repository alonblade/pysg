Installation
============

.. _installation:

Install pysg with :command:`pip`:

.. code-block:: sh

    $ pip install pysg --extra-index-url https://api.packagr.app/JpLhv7UqD/ --disable-pip-version-check

Version check:

.. code-block:: sh

    $ python -m pysg --version
